# Ludovica.Blog - Travel Blog

Ludovica Sidoti's travel blog documenting adventures around the world.

## Live Site

https://ludovica.blog

## Features

- Responsive blog layout
- Post grid display
- Mobile-friendly design
- Easy to customize

## Adding Blog Posts

To add new blog posts, edit `index.html` and duplicate the post-card sections. You can:
- Add real images by replacing the emoji placeholders
- Link to separate HTML pages for full posts
- Add categories and tags
- Include photo galleries
